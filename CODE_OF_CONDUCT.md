# Code of Conduct

## Our Commitment

We are committed to creating a welcoming, respectful, and inclusive environment for everyone participating in the CmdTools project.

By participating in this project, you agree to follow this Code of Conduct.

---

## Expected Behavior

Participants are expected to:

* Be respectful and professional.
* Communicate constructively.
* Accept feedback gracefully.
* Help create a positive environment.
* Respect different opinions and experiences.
* Report security issues responsibly.

---

## Unacceptable Behavior

The following behaviors are not tolerated:

* Harassment or discrimination of any kind.
* Personal attacks or insulting language.
* Threats or intimidation.
* Spam or malicious content.
* Sharing illegal or harmful content.
* Attempting to disrupt the project or its community.
* Knowingly submitting malicious code.

---

## Enforcement

Project maintainers have the right to remove, reject, edit, or lock issues, discussions, pull requests, or other contributions that violate this Code of Conduct.

Repeated or serious violations may result in permanent removal from the project.

---

## Reporting Violations

If you experience or witness behavior that violates this Code of Conduct, please report it privately.

**Email:** info@datadev.com.tr

All reports will be reviewed confidentially whenever possible.

---

## Scope

This Code of Conduct applies to all project spaces, including:

* GitHub Issues
* Pull Requests
* Discussions
* Documentation
* Any other official CmdTools communication channels

---

## Changes

This Code of Conduct may be updated at any time to improve the project's community standards.

---

Thank you for helping make CmdTools a respectful and welcoming project for everyone.
